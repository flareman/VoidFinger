Before executing the scripts, a certain number of command-line executables must be present in the folder along with them. A list follows:

apbs            The APBS core executable, along with all the needed Python files and libraries
                http://www.poissonboltzmann.org/apbs
            
convert         The convert program, found at the RPly library site
                http://www.impa.br/~diego/software/rply

kmltest         A CLI for the filtering algorithm, written by D. Mount
                http://www.cs.umd.edu/~mount/Projects/KMeans/
            
msms            Michael Sanner's MSMS executable
                http://mgltools.scripps.edu/downloads

polymender-dc   Tao Ju's Polymender
                http://www1.cse.wustl.edu/~taoju/code/polymender.htm

VoidFinger.jar  The core application .jar file, compiled from source using NetBeans or any other Java IDE
